## 파이썬 코딩 테스트 준비용

### 📌 목적
- 파이썬 기반 코딩 테스트 문제 해결 능력 향상
- 알고리즘/자료구조 개념을 문제 풀이를 통해 체계적으로 정리
- 대학원 진학 및 기술 면접 대비용 문제 풀이 기록

## 📏 학습 규칙
- 주 5회 이상 문제 풀
- 문제 풀이 후 반드시:
  - 접근 아이디어 정리
  - 시간 복잡도 분석
  - 실패한 풀이가 있다면 함께 기록
- 답안 참고 시, 최소 30분 이상 고민 후 확인

## 📚 학습 자료
- 교재: Do it! 알고리즘 코딩 테스트 파이썬
- 문제 플랫폼:
  - BOJ (Baekjoon Online Judge)
  - Programmers

## 🗓️ 학습 현황
- [ ] 1주차 10문제
- [ ] 2주차 5문제

## 📊 진행 상황
- 시작일: 2026.01.13
- 목표: 매주 5문제 이상
- 현재까지 풀이한 문제 수: 15

## 🚀 향후 계획
- 유형별 문제 정리 노트 추가
- 자주 틀리는 패턴 모음 정리
- 시간 제한 실패 문제 리팩토링
